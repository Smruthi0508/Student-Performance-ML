# Student Performance Prediction

Machine learning project built with Python, Pandas, NumPy and Scikit-learn to predict student exam performance.

## Features
- Data analysis with Pandas
- Visualization using Matplotlib
- Linear Regression prediction model
- Jupyter Notebook workflow

## Tech Stack
Python  
NumPy  
Pandas  
Scikit-learn  
Matplotlib

## Run the Project

```bash
pip install -r requirements.txt
jupyter notebook
